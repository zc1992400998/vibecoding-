# 基础信息展示网页规范

## 1. 项目概述

- **项目名称**: 个人信息展示页
- **项目类型**: 单页静态网页
- **核心功能**: 展示个人简介、技能、项目作品等基础信息
- **技术栈**: HTML5 + CSS3 + Vanilla JavaScript

## 2. 视觉规范

### 布局结构
- 顶部导航栏（固定）
- Hero 区域（个人介绍 + 头像）
- 关于我区域
- 技能展示区域
- 项目作品区域
- 联系方式区域
- 页脚

### 色彩方案
- 主色调: `#2c3e50`（深蓝灰）
- 强调色: `#3498db`（蓝色）
- 背景色: `#f8f9fa`（浅灰）
- 文字色: `#333333`
- 卡片背景: `#ffffff`

### 字体
- 标题: "Noto Sans SC", sans-serif
- 正文: "Noto Sans SC", sans-serif
- 代码: "Fira Code", monospace

### 动效
- 导航滚动高亮
- 卡片悬停上浮效果
- 技能条动画加载
- 平滑滚动

## 3. 功能规范

### 交互功能
- 导航栏点击平滑滚动到对应区域
- 技能展示进度条动画
- 项目卡片悬停效果
- 联系表单验证（前端）

### 内容展示
- 个人头像（使用占位图）
- 个人简介文字
- 技能列表（含熟练度百分比）
- 项目卡片（含标题、描述、标签）
- 社交链接

## 4. 验收标准

- [ ] 页面在现代浏览器中正常显示
- [ ] 导航栏固定在顶部
- [ ] 点击导航链接平滑滚动
- [ ] 技能进度条有动画效果
- [ ] 卡片悬停有视觉反馈
- [ ] 响应式布局（移动端适配）
